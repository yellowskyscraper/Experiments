Author:
nakednous A GLInteractiveFrame is a GLFrame that can be rotated and translated using the mouse.

It converts the mouse motion into a translation and an orientation updates. A GLInteractiveFrame is used to move an object in the scene. Combined with object selection, its GLMouseGrabber properties and a dynamic update of the scene, the GLInteractiveFrame introduces a great reactivity in your applications.

Method Details

addInMouseGrabberPool

public void addInMouseGrabberPool()
Adds the GLMouseGrabber in the mouseGrabberPool().

All created GLMouseGrabber are automatically added in the mouseGrabberPool() by the constructor. Trying to add a GLMouseGrabber that already isInMouseGrabberPool() has no effect.

Use removeFromMouseGrabberPool() to remove the GLMouseGrabber from the list, so that it is no longer tested with checkIfGrabsMouse(int,int,GLCamera) by the GLGraphics, and hence can no longer grab mouse focus. Use isInMouseGrabberPool() to know the current state of the GLMouseGrabber.

checkIfGrabsMouse

public void checkIfGrabsMouse(int x,

                              GLCamera camera)
Implementation of the GLMouseGrabber main method.

The GLInteractiveFrame grabsMouse() when the mouse is within a 10 pixels region around its GLCamera.projectedCoordinatesOf(PVector) position().

clearMouseGrabberPool

public void clearMouseGrabberPool()
Clears the mouseGrabberPool().

Use this method only if it is faster to clear the mouseGrabberPool() and then to add back a few GLMouseGrabbers than to remove each one independently. Use GLGraphics.setMouseTracking(false) instead if you want to disable mouse grabbing.

isSpinning

public final boolean isSpinning()
Returns true when the GLInteractiveFrame is spinning.

During spinning, spin() rotates the GLInteractiveFrame by its spinningQuaternion() at a frequency defined when the GLInteractive startSpinning().

Use startSpinning() and stopSpinning() to change this state. Default value is false.

rotationSensitivity

public final float rotationSensitivity()
Returns the influence of a mouse displacement on the GLInteractiveFrame rotation.

Default value is 1.0. With an identical mouse displacement, a higher value will generate a larger rotation (and inversely for lower values). A 0.0 value will forbid GLInteractiveFrame mouse rotation (see also constraint()). See also setRotationSensitivity(), translationSensitivity(), spinningSensitivity() and wheelSensitivity().

spinningQuaternion

public final GLQuaternion spinningQuaternion()
Returns the incremental rotation that is applied by spin() to the GLInteractiveFrame orientation when it isSpinning().

Default value is a null rotation (identity GLQuaternion). Use setSpinningQuaternion() to change this value.

The spinningQuaternion() axis is defined in the GLInteractiveFrame coordinate system. You can use GLFrame.transformOfFrom() to convert this axis from an other Frame coordinate system.

spinningSensitivity

public final float spinningSensitivity()
Returns the minimum mouse speed required (at button release) to make the GLInteractiveFrame spin().

See spin(), spinningQuaternion() and startSpinning() for details.

Mouse speed is expressed in pixels per milliseconds. Default value is 0.3 (300 pixels per second). Use setSpinningSensitivity() to tune this value. A higher value will make spinning more difficult (a value of 100.0 forbids spinning in practice). See also setSpinningSensitivity(), translationSensitivity(), rotationSensitivity() and wheelSensitivity().

startSpinning

public void startSpinning(int updateInterval)
Starts the spinning of the GLInteractiveFrame.

This method starts a timer that will call spin() every updateInterval milliseconds. The GLInteractiveFrame isSpinning() until you call stopSpinning().

stopSpinning

public final void stopSpinning()
Stops the spinning motion started using startSpinning(). isSpinning() will return false after this call.

translationSensitivity

public final float translationSensitivity()
Returns the influence of a mouse displacement on the GLInteractiveFrame translation.

Default value is 1.0. You should not have to modify this value, since with 1.0 the GLInteractiveFrame precisely stays under the mouse cursor.

With an identical mouse displacement, a higher value will generate a larger translation (and inversely for lower values). A 0.0 value will forbid GLInteractiveFrame mouse translation (see also constraint()).

Note: When the GLInteractiveFrame is used to move a Camera (see the GLInteractiveCameraFrame class documentation), after zooming on a small region of your scene, the camera may translate too fast. For a camera, it is the GLCamera.revolveAroundPoint() that exactly matches the mouse displacement. Hence, instead of changing the translationSensitivity(), solve the problem by (temporarily) setting the GLCamera.revolveAroundPoint() to a point on the zoomed region). See also setTranslationSensitivity(), rotationSensitivity(), spinningSensitivity() and wheelSensitivity().
