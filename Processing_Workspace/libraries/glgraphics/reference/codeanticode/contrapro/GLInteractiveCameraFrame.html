flyUpVector

public PVector flyUpVector()
Returns the up vector used in fly mode, expressed in the world coordinate system.

Fly mode corresponds to the GLGraphics.MOVE_FORWARD and GLGraphics.MOVE_BACKWARD GLGraphics.MouseAction bindings. In these modes, horizontal displacements of the mouse rotate the GLInteractiveCameraFrame around this vector. Vertical displacements rotate always around the Camera X axis.

Default value is (0,1,0), but it is updated by the Camera when set as its GLCamera.frame(). GLCamera.setOrientation() and GLCamera.setUpVector()) modify this value and should be used instead. TODO: fix the doc!

mouseWheelEvent

public void mouseWheelEvent(MouseWheelEvent event,
                            GLCamera camera)
Overloading of codeanticode.contrapro.GLInteractiveFrame#mouseWheelEvent(MouseWheelEvent, GLCamera).

The wheel behavior depends on the wheel binded action. Current possible actions are ZOOM, MOVE_FORWARD, MOVE_BACKWARD. ZOOM speed depends on wheelSensitivity() MOVE_FORWARD and MOVE_BACKWARD depend on flySpeed().See GLGraphics.setWheelBinding() to customize the binding. TODO: fix doc!

revolveAroundPoint

public PVector revolveAroundPoint()
Returns the point the GLInteractiveCameraFrame revolves around when rotated.

It is defined in the world coordinate system. Default value is (0,0,0).

When the GLInteractiveCameraFrame is associated to a GLCamera, GLCamera.revolveAroundPoint() also returns this value.
