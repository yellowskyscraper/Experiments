public class GLWorldConstraint
extends GLAxisPlaneConstraint
Author:
pierre A GLAxisPlaneConstraint defined in the world coordinate system.

The {@link #translationConstraintDirection()} and {@link #rotationConstraintDirection()} are expressed in the GLFrame world coordinate system (see {@link codeanticode.contrapro.GLFrame#referenceFrame()}).

Method Summary

GLQuaternion
constrainRotation(GLQuaternion rotation, GLFrame frame)
When rotationConstraintType() is Type.AXIS, constrain rotation to be a rotation around an axis whose direction is defined in the GLFrame world coordinate system by rotationConstraintDirection().
PVector
constrainTranslation(PVector translation, GLFrame frame)
Depending on translationConstraintType(), constrain translation to be along an axis or limited to a plane defined in the GLFrame world coordinate system by translationConstraintDirection().
