fromRotationMatrix

public final void fromRotationMatrix(m[][] )
Set the GLQuaternion from a (supposedly correct) 3x3 rotation matrix.

The matrix is expressed in European format: its three columns are the images by the rotation of the three vectors of an orthogonal basis. Note that OpenGL uses a symmetric representation for its matrices.

fromRotatedBasis(PVector,PVector,PVector) sets a GLQuaternion from the three axis of a rotated frame. It actually fills the three columns of a matrix with these rotated basis vectors and calls this method.

Parameters:

inverse

public final GLQuaternion inverse()
Returns the inverse GLQuaternion (inverse rotation).

The result has a negated axis() direction and the same angle().

A composition of a GLQuaternion and its inverse() results in an identity function. Use invert() to actually modify the GLQuaternion.

inverseArray

public final float[] inverseArray()

inverseMatrix

public final float[][] inverseMatrix()
Returns the associated 4x4 OpenGL inverse rotation matrix. This is simply the matrix() of the inverse().

Attention: The result is only valid until the next call to inverseMatrix(). Use it immediately (as in gl.glMultMatrixd(q.inverseMatrix())).

The matrix is given in OpenGL format (row-major order) and is the transpose of the actual mathematical European representation. Consider using inverseRotationMatrix() instead.

See Also:

slerp

public static final GLQuaternion slerp(GLQuaternion a,
                                       GLQuaternion b,
                                       float t,
                                       boolean allowFlip)
Returns the slerp interpolation of quaternions a and b, at time t.

t should range in [0,1]. Result is a when and b when t=1.

When allowFlip is true (default) the slerp interpolation will always use the "shortest path" between the quaternions' orientations, by "flipping" the source GLQuaternion if needed (see negate()).

Parameters:
a - the first GLQuaternion
b - the second GLQuaternion
t - the t interpolation parameter
allowFlip - tells whether or not the interpolation allows axis flip

squad

public static final GLQuaternion squad(GLQuaternion a,
                                       GLQuaternion tgA,
                                       GLQuaternion tgB,
                                       GLQuaternion b,
                                       float t)
Returns the slerp interpolation of the two quaternions a and b, at time t, using tangents tgA and tgB.

The resulting GLQuaternion is "between" a and b (result is a when t=0 and b for t=1).

Use squadTangent(GLQuaternion,GLQuaternion,GLQuaternion) to define the GLQuaternion tangents tgA and tgB.

Parameters:
a - the first GLQuaternion
tgA - the first tangent GLQuaternion
tgB - the second tangent GLQuaternion
b - the second GLQuaternion
t - the t interpolation parameter
