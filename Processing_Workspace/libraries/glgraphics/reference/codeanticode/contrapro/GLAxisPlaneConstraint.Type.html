public class GLAxisPlaneConstraint.Type
extends java.lang.Object
Type lists the different types of translation and rotation constraints that are available.

It specifies the meaning of the constraint direction (see translationConstraintDirection() and rotationConstraintDirection()): as an axis direction or a plane normal. GLAxisPlaneConstraint.Type means no constraint while GLAxisPlaneConstraint.Type completely forbids the translation and/or the rotation. Attention: The GLAxisPlaneConstraint.Type Type is not valid for rotational constraint.

New derived classes can use their own extended enum for specific constraints.
