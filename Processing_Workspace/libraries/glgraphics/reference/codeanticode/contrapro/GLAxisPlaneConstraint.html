Author:
pierre An abstract class for GLFrame GLConstraints defined by an axis or a plane.

GLAxisPlaneConstraint is an interface for (translation and/or rotation) GLConstraint that are defined by a direction. {@link #translationConstraintType()} and {@link #rotationConstraintType()} define how this direction should be interpreted: as an axis or as a plane normal.

The three implementations of this class: GLLocalConstraint, GLWorldConstraint and GLCameraConstraint differ by the coordinate system in which this direction is expressed.

Method Details

rotationConstraintDirection

public PVector rotationConstraintDirection()
Returns the axis direction used by the rotation constraint.

This direction is defined only when rotationConstraintType() is GLAxisPlaneConstraint.Type.

The GLAxisPlaneConstraint derived classes express this direction in different coordinate system (camera for GLCameraConstraint, local for GLLocalConstraint, and world for GLWorldConstraint). This value can be modified with setRotationConstraintDirection(PVector).

setRotationConstraintType

public void setRotationConstraintType(GLAxisPlaneConstraint.Type type)
Set the Type of the rotationConstraintType(). Default is GLAxisPlaneConstraint.Type.

Depending on this value, the GLFrame will freely rotate (GLAxisPlaneConstraint.Type), will only be able to rotate around an axis (GLAxisPlaneConstraint.Type), or will not able to rotate at all GLAxisPlaneConstraint.Type.

Use GLFrame.setOrientation(GLQuaternion) to define the orientation of the constrained GLFrame before it gets constrained.

Attention: An GLAxisPlaneConstraint.Type Type is not meaningful for rotational constraints and will be ignored.

translationConstraintDirection

public PVector translationConstraintDirection()
Returns the direction used by the translation constraint.

It represents the axis direction (GLAxisPlaneConstraint.Type) or the plane normal (GLAxisPlaneConstraint.Type) depending on the translationConstraintType(). It is undefined for (GLAxisPlaneConstraint.Type) or (GLAxisPlaneConstraint.Type).

The GLAxisPlaneConstraint derived classes express this direction in different coordinate system (camera for GLCameraConstraint, local for GLLocalConstraint, and world for GLWorldConstraint). This value can be modified with setRotationConstraintDirection(PVector).

translationConstraintType

public GLAxisPlaneConstraint.Type translationConstraintType()
Returns the translation constraint Type.

Depending on this value, the GLFrame will freely translate (GLAxisPlaneConstraint.Type), will only be able to translate along an axis direction (GLAxisPlaneConstraint.Type), will be forced to stay into a plane (GLAxisPlaneConstraint.Type) or will not able to translate at all (GLAxisPlaneConstraint.Type).

Use GLFrame.setPosition(PVector) to define the position of the constrained GLFrame before it gets constrained.
